    QIACHIP KR2202 Manual AC 110V 220V 433Mhz RF Remote Control Switch 2-CH Relay Receiver

    • Version number: V1.0
    • Compilation date: 2025/4/24
    • Model: KR2202

    Component Description

    • 1: Learning button
    • 2: Indicator light
    • 3: Antenna
    • L: Live wire
    • NO: Normally open terminal
    • NC: Normally closed terminal
    • COM: Common terminal
    • N: Neutral wire

    Function description and setting method

    Note

    1. When you match the second remote, you don't need to press the button on the receiver 8 times again to reset it.
    2. After the receiver and transmitter are paired and the working mode is set, it will retain the corresponding working mode of the transmitter regardless of whether the receiver is powered off.
    3. All the following working modes need to be implemented with QIACHIP brand remote control (transmitter) and controller (receiver / wireless remote control switch). There is no guarantee that remote controls from other brands will work properly.

    Reset function

    When the KR2202 receiver is reset, all paired transmitters will not be able to control it.

    How to Reset

    Click the receiver's learning button 8 times and when its LED flashes and then turns off, the reset will be complete.

    Description of the three working modes

    Working modes: (1) Momentary mode, (2) Toggle mode, (3) Latching mode;

    Note

    1. When you use the third working mode, you need a remote control with at least three buttons.
    2. When you use the first and second working modes, you need a remote control with at least two buttons.

    (1) Momentary mode

    In this mode:

    • Press and hold the remote control button (such as A), and the corresponding relay on the receiver is turned on.
    • Release the remote control button (such as A), and the corresponding relay on the receiver will turn off.

    How to set momentary mode

    1. Click the learning button of the receiver once. The indicator light on the receiver goes out and the receiver enters the setting state.
    2. Press the button on the remote control (such as A) once. The LED indicator on the receiver will flash and then turn off.
    3. At this time, press another button (such as button B) of the same remote control. The indicator light on the receiver flashes and then turns on. The momentary mode is set successfully.

    (2) Toggle mode

    In this mode:

    • Press the remote control button (such as button A), and the corresponding relay on the receiver will turn on.
    • Press the remote control button (such as button A) again, and the corresponding relay on the receiver will turn off.

    How to set toggle mode

    1. Click the learning button of the receiver twice. The indicator light on the receiver goes out, and the receiver enters the setting state.
    2. Press the button on the remote control (such as A) once. The LED indicator on the receiver will flash and then turn off.
    3. At this time, press another button (such as button B) of the same remote control. The indicator light on the receiver flashes and then turns on. The toggle mode is set successfully.

    (3) Latching mode

    In this mode:

    • Press the remote control button (such as A), and the corresponding receiver's relay A is turned on.
    • Press the remote control button (such as B), and the receiver's relay B is turned on, and the receiver's relay A is turned off.
    • Press the button C on the remote control, and the turned-on relay on the receiver will be turned off.

    How to set latching mode

    1. Click the learning button of the receiver three times. The indicator light on the receiver goes out, and the receiver enters the setting state.
    2. Press the button on the remote control (such as A) once. The LED indicator on the receiver will flash and then turn off.
    3. At this time, press another button (such as button B) of the same remote control. The indicator light on the receiver flashes and then turns on. The Latching mode is set successfully.

    Electrical characteristics

    Parameter Value
    Input voltage AC 85V-250V
    RF frequency 433.92MHz
    Standby current 8.5mA
    Load power 1100W
    Receiver sensitivity -108dbm
    Operation mode Momentary mode/Toggle mode/Latching mode
    Working temperature -10℃~70℃
    Size 68×48×18mm

    Warning

    • L and N lines cannot be reversed.
    • Be safe and always keep the power off when wiring.
    • When using wireless electronics, take care to avoid metal objects, large electronic devices, electromagnetic fields and other sources of strong interference.
